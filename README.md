## OSEAN bridge UI 
To use with Socket.tech bridge contracts SuperTokenVault and SuperToken. 

## Getting Started

- Clone this repository
- Install dependencies

```bash
yarn install
```

- Create .env and get your API keys from Thirdweb

## Dev enviroment

```bash
yarn dev
```

## Build

```bash
yarn build
```

## Production

```bash
yarn start
```
